<h4>Formula used :</h4><br/>

<h3>
<blockquote cite="https://www.bajajhousingfinance.in/how-to-calculate-home-loan-emi">
EMI = (P * r * (1 + r)^n)/ (((1 + r)^n) - 1) <br/>
where r is rate/(12*100)
</blockquote>
</h3>
