
# Digital Storage Calculator

Digital storage Calculator will be used to inter convert value of different units of Digital storage into each other.


## Supported Storage Units :

- BIT
- BYTE
- KILOBIT
- KILOBYTE
- MEGABIT
- MEGABYTE
And Many More ..


## Built with

 - HTML
 - CSS
 - Javascript
