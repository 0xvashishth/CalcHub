# Number System Conversion

## Descriptions :

> Convert Number in to other formate like as binary, octal and hex.

## Functionality :
> able to convert number in to following types

```
> Decimal To Binary
> Decimal To Octal
> Decimal To Hex

> Binary To Decimal
> Binary To Octal
> Binary To Hex

> Octal To Decimal
> Octal To Binary
> Octal To Hex

> Hex To Decimal
> Hex To Binary
> Hex To Octal
```
