
# Currency Calculator

The provided Currency Calculator is capable of converting value of one Currency unit into another Currency unit.


## Supported Currency :

- USD
- INR
- HKD
- USD
- EUR
- JPY
And Many More ..


## Built with

 - HTML
 - CSS
 - Javascript
